Creating Portfolios

In Swatch its super easy to create portfolios and make them look unique. You can use multiple portfolio types (image, gallery and videos) and use different page templates for each portfolio item. As anything else in Swatch each portfolio item can have its own color. Letâ€™s get started.

Portfolio Item Preview

Lets take a look at how a portfolio item looks like.

  1. The title of the portfolio item.
  2. The portfolio item content.
    NOTE! All the content before the tag is used as a caption when hovering the portfolio item in a 2, 2 or 4 column portfolio list. The content after the shortcode will appear in the single portfolio page as content.
  3. The portfolio post format.
  4. The Categories that the portfolio belongs to.
  5. The color Swatch for the portfolio item.
  6. The skills related with the portfolio item.
  7. The portfolio item template, Bigor Small.
  8. A link (url) related with the portfolio item.
  9. Portfolio description field. Any content added here will appear in single portfolio before the content.
  10. The order of appearance of the portfolio item (in the portfolio list).
  11. Portfolio featured image (always to include one).

Creating Portfolio Items

Take a look at the video to see how you can add portfolio items.

By default the page headers are not showing. You can set the page headers in the Header options panel.

Portfolio Types

Swatch supports 3 different portfolio post types: standard(image only), gallery(gallery of images) and video. Take a look at the screencast to see how you can set up the portfolio type.

TIP! To link link a portfolio type to any content (post page or even extarnal url) select the link type from the link option inside the portfolio item.

Portfolio Templates

In Swatch you can use 2 different page templates for your portfolio items. Big or small image. Have a look at the screencast to see how the templates are working.

Adding portfolios to pages

In this screencast you can see how you can add a portfolio on a page.

NOTE! As of version 1.3 you can add pagination in the portfolio list. Just enable the Use pagination option in the shortcode and select the number of portfolio items per page.
