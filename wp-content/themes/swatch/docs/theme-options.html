Logo

These options allow you to configure the site logo, you can select a logo type and then create a text logo, image logo or both image and text. There is also an option to use retina sized images.

Name Description Default
Logo Type Select which kind of logo you would like.
Text - Plain text only logo (uses body font)
Image - Image only logo
Text & Image - Text followed by an image (text will use the body font)
options : Text, Image, Text & Image 		text
Logo Text Add your logo text here works with Logo Type (Text, Text & Image) Swatch
Logo Image Upload a logo for your site, works with Logo Type (Image, Text & Image)
Retina Logo Use retina logo (NOTE - you will need to upload a logo that is twice the size intended to display)
options : Retina Logo, Normal Logo 		off
