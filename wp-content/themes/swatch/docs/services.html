Creating Services

One of the most powerful features in Swatch theme is the ability to add and customize the services that your company can offer. In this section we’ll see how we can add and showcase services.

The Anatomy of a Service

Service Backend

  1. The service title.
  2. Type the Service intro content that services shortcode will display, insert a ‘more’ and a ‘noteaser’ tag and then type the service main content that will appear in the single service page:
     <!--more--><!-noteaser-->
  3. Group the service to an existing category/add a new category on the spot.
  4. Enter the order of the service.
  5. Add a feature image. That will show in the circle.
  6. Select the icon you want use (can be left blank).
  7. Select a CSS3 animation for your icon.
  8. Select if you want to use the feature image (options are, “feature image”, “icon”, “image and icon”, “nothing”).
  9. Select the page header options for the single service page.
  10. Select where you want the link on the services list to take you (options are “default link” that will redirect you to the single service page, “page”, “post”, “portfolio”, “category” and “url”). In that way the link when you click on the service title can take you pretty much anywhere you wish.
  11. Select the link behaviour (“Same page as it was clicked”, “Open in new window”, “Open the linked document in the parent frameset”or “Open the linked document in the full body of the window”).
  12. Select the page template of the single service (“fullwidth”, “left sidebar” or “right sidebar”)
