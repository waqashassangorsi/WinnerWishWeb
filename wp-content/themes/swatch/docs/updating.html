Envato Toolkit Plugin

Envato Toolkit plugin is deprecated, use the Envato Market plugin instead

Envato Market Plugin

Updating is a lot easier if you use the Envato Market. The Envato Market plugin can install WordPress themes and plugins purchased from ThemeForest & CodeCanyon by connecting with the Envato Market API using a secure OAuth personal token. Once your themes & plugins are installed, WordPress will periodically check for updates, so keeping your items up to date is as simple as a few clicks.

You can add a global token to connect all your items from your account, and/or connect directly with a specific item using a singe-use token & item ID. When the global token and single-use token are set for the same item, the single-use token will be used to communicate with the API.

Instructions

Here are the step by step instructions on how to update your theme

Step 1 - Install the Envato Market plugin - go to Appearance/ Install Pluginsand Install and Activate the Envato Market Plugin.

Step 2 - Click the Envato Marketmenu on the admin sidebar.

Step 3 - Enter your Envato API Personal Token. If you don't have an API token read the create an api key section

Create a ThemeForest API Key

To get your copy of the theme follow the instructions below.

Step 1 - Log in to your themeforest account and then goto the account menuand click the Settingsoption from the dropdown

Step 2 - Click the API Keys menu button at the bottom of the left side menu

Step 2 - Click the API Keys menu button at the bottom of the left side menu

Step 3 - Enter a label for your API key and then click the Generate API Keybutton

Manual Update

If you want to manually update the theme without using the Envato Toolkit Plugin you will need to do the following.

Step 1 - Switch to another theme - goto Appearance/ Themesand select activate on another theme.

Step 2 - Hover over the Swatch screenshot and click the Theme Detailsbutton

Step 3 - Click the Deletebutton in the bottom right hand corner of the popup

Step 4 - Follow the installing the theme instructions to re-install the latest version of the theme.
