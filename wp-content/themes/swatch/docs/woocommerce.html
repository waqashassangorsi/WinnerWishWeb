Installing WooCommerce plugin

You can download the Woocommerce plugin from here. You can then install it from the Plugins->Add New menu as you would install any plugin. Alternatively, you can install it from Appearance->Install Plugins. Once installed, activate it from the same page.

Setting shop swatch

You can set the shop swatch from Swatch->WooCommerce, from the Shop Optionssection, by setting the General Shop Swatchoption.

Setting shop layout

You can set the shop layout from Swatch->WooCommerce, from the Shop Optionssection, by setting the Shop Layoutoption.

Product list pages

Setting category page header

You can set a product category's header by editing that category and changing the name of that category

Setting tag page header

You can set a product tag's header by editing that tag and changing the name of that tag

Product pages

Setting shop social links

You can set the social links for the shop from Swatch->WooCommerce, from the Shop Optionssection, by setting the Social Icons In Product Detailsoption to On/Off.

Changing product slideshow options

You can set the product slideshow options for the shop from Swatch->WooCommerce, from the Product Slideshowoptions section.

  • Animation style: Select how your slider animates.
  • Speed: Set the speed of the slideshow cycling, in milliseconds.
  • Duration: Set the speed of animations.
  • Auto start: Start slideshow automatically.
  • Show navigation arrows: Show or hide the navigation arrows.
  • Navigation arrows type: Type of the direction arrows, fancy (with bg) or simple.
  • Show controls: If you choose hide the option below will be ignored.
  • Choose the place of the controls: Choose the position of the navigation controls.
  • Choose the alignment of the controls: Choose the alignment of the navigation controls.

Setting shop pages

You can set the page that you want to use as your shop page from Woocommerce->Settings, under the Productstab. From the Product Listingssection, set the Product Archive / Shop Pageoption to be the page that you want to use as shop page.

Also, you can set your checkout pages from Checkouttab, from the Checkout Pagesoptions section. Set a page for the Cart, Checkout Pageand Terms and Conditions. In addition, you can set your Checkout endpointsfrom the same page.

For more info about the Woocommerce endpoints, visit the WooCommerce Documentation on endpoints

Similarly, you can set the Accountpage and endpoints from the Accountstab.

Setting account page templates

The account page can have a menu on the left side of the page, that helps the user navigate through his account settings easier. To achieve this, you need to assign the Woocommerce account templateto the page that you use as account page. To do that, edit that page and from the Page Attributesmetabox under the Updatebutton, set the aforementioned template.
