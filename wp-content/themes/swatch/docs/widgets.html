Widgets

Swatch provides support for all the wordpress native widgets. Additionally Swatch offers 3 custom widgets for social icons, WPML language selector and twitter feeds.

Social Icon Widget

In the social icons widget you can set:

  1. If the social links will open in a new window/tab.
  2. If you want the icons to be placed in a coloured background.
  3. If you want the icon size to be big or small.
  4. Set the icon and the url for each social link.
The icons are from the collection of fontawesome icons

For the headers widgets use the mini size in order to fit the height!

WPML Language Selector Widget

In this widget you can set up

  1. If you want the language to be displayed as name, flag or name and flag
  2. The appearance order of the languages.
  3. The order direction.
  4. If you want to skip missing languages.

Twitter Feed Widget

In this widget you can set up

  1. The username of the tweet feed.
  2. The consumer key.
  3. The consumer secret.
  4. The access token.
  5. The access token secret.
  6. The maximum number of tweets.
  7. Various options on how the tweets are displaying.
