Revolution Slider

Installing revolution slider plugin

Revolution Slider plugin comes bundled with the theme. If you go to Appearance->Install Plugins, you can install it from the list of the required plugins. Once installed, activate it.

Creating a slider

Click on the Revolution Slider menu item on the admin panel. You can create a new slider if you click on the Create New Sliderbutton on the next screen.

For more info on how to create a new slider read on the documentation of the plugin.

Importing a slider

To import a slider click on Revolution Slider on the admin sidebar. From the next screen click on the Import Sliderbutton. You will be asked to upload the import file from a new window that will popup.
When you click on the Revolution Slider menu, if a slider exists, you should see an Export Sliderbutton. Clicking on this button should download the file needed to do an import.

Updating revolution slider

Never uninstall/reinstall the plugin, as that would also delete your previously built sliders. Instead, as long as the following instructions are followed, any previously built sliders will be saved and still exist once the update process is complete.

If you wish to update Revolution Slider and the version that you have currently installed is 2.3.3 - 4.6.9, follow these steps

  1. Locate the plugin from the theme folder, inside inc/plugins. The name of the file is revslider.zip.
  2. Click on the Revolution Slider menu on the admin sidebar. Scroll down to the bottom of the next page and click on the Manual Plugin Updatebutton.
  3. Upload the file that you downloaded(revslider.zip) and click on Update Sliderfrom the window that pops up.

Your previous data will not be lost.

If you wish to update Revolution Slider and the version that you have currently installed is 4.6.91+, follow these steps

  1. Locate the plugin from the theme folder, inside inc/plugins. The name of the file is revslider.zip.
  2. From your favorite FTP program, delete the folders/files in the following directory:
    wp-content/plugins/revslider/
  3. Replace them with the folders/files located inside the “revslider” folder that can be found inside the zip file you obtained from Swatch.
Note that updating from v4.*.* to v5.*.* might result in losing your custom styling of the slides as this is a major update and the plugin has changed the way it loads the css. Make sure that you make a backup of your slides before updating.
