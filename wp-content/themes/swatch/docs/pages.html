Setting the Page Header

Swatch allows you to set up you page header for every single page. Take a look at the video for more details.

By default the page headers are not showing. You can set the page headers in the Header options panel
If you want you can completely remove the menu header on any page by selecting the 'no-header' option in the page 'Site header' options

Using Page Sections

Swatch uses Sections to make the content stand out and be easy to customise. Take a look at the video to see how you can set up page sections.

NOTE! Always add your content inside a Section shortcode

Page Templates

Swatch comes with 3 page templates, full width(default) left sidebarand right sidebar.

NOTE! Always use Section sortcodes when using the default (fullwidth) template
